# Get stating with react project

This is project have backend and frontend

## Install module

`yarn`

## Run project

`yarn start`
or dev mode in backend
`yarn start`

## Run frontend

`yarn start:app`

## Run backend

`yarn start:be`
or dev mode
`yarn dev:be`
